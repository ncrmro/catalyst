# Rails Blog Boilerplate

This is a simple Rails blog application generated as a boilerplate for the Catalyst project.

## Features

* Article CRUD operations (Create, Read, Update, Delete)
* SQLite database
* Rails 8.0 with modern defaults
* Test suite included
